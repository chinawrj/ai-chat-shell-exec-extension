# AI Chat Shell Exec v0.9.2

This patch fixes a v0.9.1 regression where page-driven composer redraws could leave plugin-inserted content unsent.

## What changed

- Helper-output and agent-prompt delivery now reacquire the current visible composer when the page replaces the original writer node.
- Ownership transfers only when the replacement contains the exact plugin-inserted text; no second composer write occurs.
- Empty or different composer text remains user cancellation and is never adopted, overwritten, or sent.
- Hidden or detached stale composer nodes are no longer eligible for auto-send.

## Validation before release

- Focused helper-output redraw regression passed.
- Focused agent-prompt redraw regression passed.
- Intentional deletion, unrelated draft, single-write, and send-retry regression suites passed.

## Post-release validation

- Full Chromium E2E and the complete test suite will be run immediately after publishing, as requested.
