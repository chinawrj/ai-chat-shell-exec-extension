# AI Chat Shell Exec v0.2.5

Manual retry hardening for `Run latest`.

## Highlights

- Keeps manual `Run latest` forced when it is clicked while a previous shell call is still clearing.
- Retries the forced scan briefly instead of falling back to ordinary automatic scanning.
- Shows `No shell-call found on this page` when the manual scan has no runnable candidate.
- Includes the v0.2.4 settings migration fix for legacy chain-limit values saved as `"5"`.

## Install Or Upgrade

1. Download and unzip `ai-chat-shell-exec-extension-v0.2.5-source.zip`.
2. Open `chrome://extensions`, enable Developer mode, and load the extracted project root or `extension/`.
3. From the extracted project root, run:

   ```sh
   ./scripts/install_shell_server_agent.sh
   ```

4. Reload enabled chat tabs and confirm the floating panel shows `Run latest`.
