# AI Chat Shell Exec v0.3.3

Default ForAI tmux workspace release.

## Highlights

- The page plugin now checks tmux startup state and asks the local server to ensure a default `ForAI` tmux session.
- The server automatically creates `host` and `board` windows inside `ForAI` when they are missing.
- Shell helpers can omit an explicit target; default commands run in the `ForAI` `host` window.
- Board helpers default to the `ForAI` `board` window, while `AI_CHAT_SHELL_BOARD_TARGET` still overrides the target.

## Install Or Upgrade

1. Download and unzip `ai-chat-shell-exec-extension-v0.3.3-source.zip`.
2. Open `chrome://extensions`, enable Developer mode, and load the extracted project root or `extension/`.
3. Restart the local shell server from the extracted project root.

   On macOS:

   ```sh
   ./scripts/install_shell_server_agent.sh
   ```

   On Ubuntu, or for a temporary foreground server:

   ```sh
   ./scripts/start_shell_server.sh
   ```

4. Reload enabled chat tabs.
5. Use the floating panel's `Check` button and confirm it reports the `ForAI` host target.
