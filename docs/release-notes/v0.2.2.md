# AI Chat Shell Exec v0.2.2

Quick compatibility release for longer AI-driven command chains.

## Highlights

- The default automatic shell-call chain limit is now `100`.
- The popup no longer enforces an upper bound on `Chain limit`; values must only be at least `1`.
- Existing installs that still have the old default value `5` are migrated to `100`.
- Custom chain-limit values are preserved.

## Install Or Upgrade

1. Download and unzip `ai-chat-shell-exec-extension-v0.2.2-source.zip`.
2. Open `chrome://extensions`, enable Developer mode, and load the extracted project root or `extension/`.
3. From the extracted project root, run:

   ```sh
   ./scripts/install_shell_server_agent.sh
   ```

4. Reload enabled chat tabs and confirm the floating panel appears.
