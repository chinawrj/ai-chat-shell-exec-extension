# AI Chat Shell Exec v0.5.2

Fenced helper end-marker recovery release.

## Highlights

- Closed four-backtick fenced shell helpers can now run even when the AI forgets `ai-helper-shell-end`.
- Closed four-backtick fenced file helpers can now write files even when the AI forgets `ai-helper-file-end`.
- Closed four-backtick fenced board helpers can now send board commands even when the AI forgets `ai-helper-board-end`.
- The fallback only applies when the helper start marker is inside a complete four-backtick fence. Incomplete helper starts still do not execute.
- Normal explicit `ai-helper-*-end` markers remain supported and take priority.

## Upgrade Notes

1. Download and unzip `ai-chat-shell-exec-extension-v0.5.2-source.zip`.
2. Reload the unpacked extension from the matching `extension/` directory.
3. Stop any older foreground shell server.
4. Start the current server with `./scripts/start_shell_server.sh`.
5. Use the popup or floating-panel `Check` action and confirm it reports server protocol `3`, helper `1`, and `ForAI` readiness.
