# AI Chat Shell Exec v0.9.3

This patch adds the missing durable send fallback for every helper result.

## What changed

- Shell, board, file, agent-message, roster, and task-status results all enter the same bounded local delivery queue after the backend returns.
- A failed first send attempt retries only button/form/keyboard submission. The backend operation and composer write remain single-shot.
- Exact text may transfer to a current visible composer after a page redraw. Empty or different user text cancels and is never overwritten or sent.
- The real Chromium file-helper test now requires a submitted user message after synchronously replacing the composer during the plugin input event.

## Validation before release

- All content-script regression suites passed.
- The new file-helper fallback test proves one backend operation, one composer write, and a successful second send attempt.
- Composer-redraw helper and agent tests passed.

## Post-release validation
