# AI Chat Shell Exec v0.5.1

Basic tmux path documentation consistency patch.

## Highlights

- Corrects `AGENTS.md` so shell helper blocks use the current no-target format and always run in `ForAI:host`.
- Keeps the non-vision tmux path explicit for users and maintainers: foreground server startup, automatic `ForAI` workspace setup, and board target handling stay separate from shell helpers.
- Adds a regression check so docs do not reintroduce the obsolete shell target line.

## Upgrade Notes

1. Download and unzip `ai-chat-shell-exec-extension-v0.5.1-source.zip`.
2. Reload the unpacked extension from the matching `extension/` directory.
3. Stop any older foreground shell server.
4. Start the current server with `./scripts/start_shell_server.sh`.
5. Use the popup or floating-panel `Check` action and confirm it reports server protocol `3`, helper `1`, and `ForAI` readiness.
