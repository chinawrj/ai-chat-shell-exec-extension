# AI Chat Shell Exec v0.1.0

Initial public release of AI Chat Shell Exec.

## Highlights

- Explicit `shell-call` tool blocks from AI chat pages.
- Local WebSocket shell server on `127.0.0.1:17371`.
- Automatic `shell-output` reply posting.
- Zero-knowledge site strategy with optional user-bound input, send, and shell display selectors.
- Floating panel with self-test and health check controls.
- Duplicate execution protection in the extension and local server.
- macOS LaunchAgent installer and uninstaller.

## Install

1. Download and unzip `ai-chat-shell-exec-extension-v0.1.0-source.zip`.
2. Open `chrome://extensions`, enable Developer mode, and load the extracted project root or `extension/`.
3. From the extracted project root, run:

   ```sh
   ./scripts/install_shell_server_agent.sh
   ```

4. Refresh the AI chat page and use the floating `Check` or `Test` buttons.

## Security

This release enables local shell execution requested by AI chat pages. Use it only on machines you control. Review `SECURITY.md` before using it with important files or credentials.
