# AI Chat Shell Exec v0.2.3

Manual retry release for stuck or intentionally repeated `shell-call` blocks.

## Highlights

- Adds a `Run latest` button to the floating panel.
- `Run latest` rechecks the current page once and executes the latest detected `shell-call`.
- This manual path ignores automatic execution limits such as processed-call tracking, duplicate-output suppression, self-test waiting, and chain limits.
- Forced runs use a one-time call key so background and server ledgers do not suppress the retry.

## Notes

- The command still needs to be a valid `shell-call` with `target` and `cmd`.
- Empty commands and missing tmux targets are still rejected because there is nothing safe to execute.

## Install Or Upgrade

1. Download and unzip `ai-chat-shell-exec-extension-v0.2.3-source.zip`.
2. Open `chrome://extensions`, enable Developer mode, and load the extracted project root or `extension/`.
3. From the extracted project root, run:

   ```sh
   ./scripts/install_shell_server_agent.sh
   ```

4. Reload enabled chat tabs and confirm the floating panel shows `Run latest`.
