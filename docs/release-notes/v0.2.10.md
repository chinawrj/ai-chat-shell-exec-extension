# AI Chat Shell Exec v0.2.10

Helper identity suffix release.

## Highlights

- Adds optional helper identity suffixes such as `ai-helper-shell-start:2` and `ai-helper-file-start:2`.
- Uses the helper identity in duplicate suppression so otherwise-identical helper payloads with different suffixes can run as separate requests.
- Keeps unsuffixed helper blocks valid by deriving a stable identity from the plain text helper payload.
- Rejects malformed suffixes and continues to reject JSON helper requests and old `ai-helper-start-shell` aliases.

## Install Or Upgrade

1. Download and unzip `ai-chat-shell-exec-extension-v0.2.10-source.zip`.
2. Open `chrome://extensions`, enable Developer mode, and load the extracted project root or `extension/`.
3. From the extracted project root, start the local shell server.

   On macOS:

   ```sh
   ./scripts/install_shell_server_agent.sh
   ```

   On Ubuntu, or for a temporary foreground server:

   ```sh
   ./scripts/start_shell_server.sh
   ```

4. Reload enabled chat tabs and confirm the floating panel appears.
