# AI Chat Shell Exec v0.2.7

Response-space release for `shell-output` blocks.

## Highlights

- Displays only a short command preview in returned `shell-output` blocks.
- Long commands are capped at 64 displayed characters before `...`.
- Multiline commands are collapsed to one display line.
- When the display is abbreviated or normalized, the block includes `cmdHash` so duplicate suppression still recognizes the original command.
- stdout and stderr are unchanged, leaving more space for the actual command response.

## Install Or Upgrade

1. Download and unzip `ai-chat-shell-exec-extension-v0.2.7-source.zip`.
2. Open `chrome://extensions`, enable Developer mode, and load the extracted project root or `extension/`.
3. From the extracted project root, run:

   ```sh
   ./scripts/install_shell_server_agent.sh
   ```

4. Reload enabled chat tabs and confirm the floating panel shows the current version.
