# AI Chat Shell Exec v0.2.6

Manual retry reliability fix.

## Highlights

- Releases the active command lock after the shell output is inserted, before waiting on chat auto-send confirmation.
- This prevents `Run latest` from being blocked by pages whose send confirmation takes longer than expected.
- Keeps the v0.2.5 forced retry queue for clicks that land while the previous command is still finishing.
- Includes the v0.2.4 one-time migration for legacy chain-limit settings.

## Install Or Upgrade

1. Download and unzip `ai-chat-shell-exec-extension-v0.2.6-source.zip`.
2. Open `chrome://extensions`, enable Developer mode, and load the extracted project root or `extension/`.
3. From the extracted project root, run:

   ```sh
   ./scripts/install_shell_server_agent.sh
   ```

4. Reload enabled chat tabs and confirm the floating panel shows `Run latest`.
