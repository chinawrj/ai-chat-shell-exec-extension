# AI Chat Shell Exec v0.1.1

Documentation-focused release that adds copy-paste AI instructions.

## Changes

- Adds `docs/AI_INSTRUCTIONS.md`.
- Adds minimal, recommended, project-agent, one-off, and manual test instruction templates.
- Updates README setup flow to tell users to configure AI instructions before running the full-chain test.

## Install

1. Download and unzip `ai-chat-shell-exec-extension-v0.1.1-source.zip`.
2. Open `chrome://extensions`, enable Developer mode, and load the extracted project root or `extension/`.
3. From the extracted project root, run:

   ```sh
   ./scripts/install_shell_server_agent.sh
   ```

4. Add one of the templates from `docs/AI_INSTRUCTIONS.md` to your AI chat system's custom, project, or agent instructions.
5. Refresh the AI chat page and use the floating `Check` or `Test` buttons.
