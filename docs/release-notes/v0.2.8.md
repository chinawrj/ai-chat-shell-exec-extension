# AI Chat Shell Exec v0.2.8

Plain helper-block release for shell and file requests.

## Highlights

- Replaces JSON `shell-call` requests with plain text `ai-helper-shell-start` / target / command / `ai-helper-shell-end` blocks.
- Adds `ai-helper-file-start` / filename / content / `ai-helper-file-end` blocks for writing a single file under `$HOME/Downloads`.
- Removes support for JSON shell-call parsing and the old `ai-helper-start-shell` / `ai-helper-end-shell` aliases.
- Updates AI-facing instructions so helper blocks are described as requests served by the human helper, not as an automatic script interface.
- Adds a real Chrome extension e2e test that loads the unpacked extension, drives the tmux test page, and verifies both shell and file helper result replies.

## Install Or Upgrade

1. Download and unzip `ai-chat-shell-exec-extension-v0.2.8-source.zip`.
2. Open `chrome://extensions`, enable Developer mode, and load the extracted project root or `extension/`.
3. From the extracted project root, run:

   ```sh
   ./scripts/install_shell_server_agent.sh
   ```

4. Reload enabled chat tabs and confirm the floating panel appears.
