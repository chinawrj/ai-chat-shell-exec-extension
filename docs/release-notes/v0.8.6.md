# AI Chat Shell Exec v0.8.6

This release adds multi-board support for board helpers while preserving the existing default board workflow.

## Changes

- Added named board helper markers such as `ai-helper-board-R1-start` / `ai-helper-board-R1-end`, which route commands to the `ForAI:board-R1` tmux window.
- Kept `ai-helper-board-start` / `ai-helper-board-end` targeting the default `ForAI:board` window.
- Preserved `AI_CHAT_SHELL_BOARD_TARGET` as the highest-priority board override for local operator control.
- Added server-side validation so named board requests must be empty or `board-<safe suffix>` and cannot select arbitrary tmux targets.
- Updated shell-output metadata, README guidance, AI-facing instructions, and the feature/test matrix for named board helpers.

## Validation

- `./scripts/test_all.sh`
- Real tmux integration coverage for default `board` and named `board-R1` prompt probing, command send, and output capture.
- Independent subagent tester validation for parser, background forwarding, server resolution, tmux integration, and default board regression.
- Independent subagent code review of the named board feature path.
