# AI Chat Shell Exec v0.2.4

Small settings-migration follow-up for the manual retry release.

## Highlights

- Keeps the `Run latest` manual retry behavior from v0.2.3.
- Migrates old default chain-limit values saved as either `5` or `"5"` to the new default `100`.
- Records the settings migration version so users can still intentionally set a lower chain limit after upgrading.

## Install Or Upgrade

1. Download and unzip `ai-chat-shell-exec-extension-v0.2.4-source.zip`.
2. Open `chrome://extensions`, enable Developer mode, and load the extracted project root or `extension/`.
3. From the extracted project root, run:

   ```sh
   ./scripts/install_shell_server_agent.sh
   ```

4. Reload enabled chat tabs and confirm the floating panel shows `Run latest`.
