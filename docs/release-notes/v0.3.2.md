# AI Chat Shell Exec v0.3.2

Four-backtick helper fence release.

## Highlights

- Updates README and AI instruction helper examples from three-backtick fences to plain four-backtick fences.
- Updates the floating-panel self-test prompt so new compatibility checks ask the AI for a four-backtick helper block.
- Keeps helper detection marker-based, so rendered `ai-helper-*` blocks still work after chat UIs strip markdown fences.
- Unifies the floating panel, extension, and release version at `0.3.2`.
- Adds a floating-panel version check that warns when the page content script and background worker versions do not match.

## Install Or Upgrade

1. Download and unzip `ai-chat-shell-exec-extension-v0.3.2-source.zip`.
2. Open `chrome://extensions`, enable Developer mode, and load the extracted project root or `extension/`.
3. Restart the local shell server from the extracted project root.

   On macOS:

   ```sh
   ./scripts/install_shell_server_agent.sh
   ```

   On Ubuntu, or for a temporary foreground server:

   ```sh
   ./scripts/start_shell_server.sh
   ```

4. Reload enabled chat tabs.
5. Copy the updated AI instructions from the README or `docs/AI_INSTRUCTIONS.md`.
6. Use the floating panel's `Check` button and confirm it reports `Extension v0.3.2` without a version mismatch.
